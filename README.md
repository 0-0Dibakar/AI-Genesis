## Getting Started

### Local Development
1. Clone the repository:
```bash
git clone https://github.com/0-0Dibakar/AI-Genesis.git
```

2. Open `index.html` in your browser:
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

### Deployment to GitHub Pages

The repository is ready to deploy on GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Select "main" branch as source
4. Save
5. Your site will be live at: `https://0-0Dibakar.github.io/AI-Genesis`

## Customization

Edit `index.html` to customize:
- **Company Name**: Replace "VentureX" with your company name
- **Metrics**: Update the hero section metrics (10K+ users, $50M+ value, etc.)
- **Team Members**: Add your actual team information
- **Contact Information**: Update email, phone, location
- **Colors**: Modify CSS variables in the `:root` section for brand colors
- **Content**: Update all section content with your startup's information
